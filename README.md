# SimpleBlackJackGame

A command-line Blackjack game written in Python. My first solo project, built without tutorials to test what I actually knew.

## How to Play

Run the game:
```bash
python blackjack.py
```

- You'll be asked if you want to **hit** (y) or **stand** (n)
- Try to get as close to **21** as possible without going over
- The computer will then play its hand automatically
- Whoever is closest to 21 wins!

## Features

- Player hit or stand logic
- Computer dealer that stops at 17 (just like real Blackjack)
- Bust detection for both player and dealer

## What I Learned

- Writing functions to separate logic
- Using `while` loops and `break` statements
- Working with `random` and user input in Python
